# 提交备注：大一统帝国调度模拟器（Codex 版）

- **来源**：本地 `airport-simulate/` 目录，由 **Codex** 生成（目录中带有 `.openai/` 配置，提交时已排除）。
- **与机场游戏的关系**：与 `第08天-Airport Control`（机场塔台画线引导）同源——同一套"实时调度、画线指派、避免冲突"机制的主题换皮：飞机起降换成了名将平定叛乱/救灾/肃贪。代码中仍保留 `landings`、`pace` 等机场游戏的变量命名痕迹。
- **玩法**：按住京城下方的空闲名将，拖动到危机圆点后松开指派；任一危机存在超过 20 秒即游戏结束。
- **技术栈**：Svelte + TypeScript + Vite + Tailwind CSS + HTML5 Canvas（工程化项目，非单文件）。
- **运行方式**：`npm install && npm run dev`。
- **说明**：`node_modules`、`dist`、`build`、`.git` 等依赖与构建产物未提交。
