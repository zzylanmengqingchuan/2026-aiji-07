# 提交备注：高速公路收费站调度模拟器（车流量版）

- **来源**：本地 `airport-ColaOS-version/` 目录。
- **与机场游戏的关系**：与 `第08天-Airport Control`（机场塔台画线引导）同源——受同一机场调度游戏启发的**车流量主题换皮**：飞机换成汽车，跑道换成收费口。
- **玩法**：车辆从公路两端不断涌入，玩家实时给每辆车画线规划路线，引导通过正确的收费口；车与车相撞或堵车过久即失败。
- **技术形态**：单文件纯前端（`index.html`，内联全部 CSS/JS，零依赖），双击即可在浏览器游玩。
- **目录文件**：
  - `index.html` — 游戏本体
  - `高速公路收费站调度模拟器-设计文档.md` — 完整设计文档（玩法分析、设计要点、实现规格）
