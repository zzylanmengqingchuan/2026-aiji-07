# 3D 坦克大战 · 4 人混战联机版（Agent 优先）

基于王树义老师公开的单机 3D 坦克大战改造：多人或 Agent 联机自由混战。

- **人类入口**：浏览器 `http://HOST:3099/`（支持 `?room=XXXX` 邀请链接）
- **Agent 入口**：`GET /api/v1/docs`，详见 [AGENT.md](./AGENT.md)
- **云上更新**：见 [DEPLOY.md](./DEPLOY.md)

## 玩法设计

| 项目 | 规则 |
|------|------|
| 模式 | 自由混战（Free-for-All），不是合作打怪 |
| 人数 | 2～8 人 |
| 赛程 | 10 局，每局 3 把，每把 60 秒 |
| 胜负 | 每把剩 1 人时提前结束；最终按全场总积分排名 |
| 对撞 | 坦克贴身碰撞会同归于尽，可快速结束当前把 |
| 出生 | 四角出生点（双人对角优先）+ 约 2 秒无敌 |
| 操作 | WASD 移动；鼠标控制炮台；Q/E 环绕镜头；滚轮调整视角高低；空格/左键开火 |
| 观战 | 二点五维/二维切换；方向键或 WASD 平移；滚轮缩放；可跟踪指定玩家 |
| 测试 | 房主或选手可提前结束，并按当前积分生成完整战报 |
| 补给 | 当前关闭，避免干扰三命积分规则 |
| 去掉 | AI 敌人和波次（避免干扰玩家对战） |

保留：同一战场手感、障碍遮挡、坦克模型风格、射击/护盾/速射等。

## 架构

```
浏览器客户端 × N
    │  WebSocket 输入
    ▼
Node 服务端（权威模拟 20 tick/s）
    │  状态快照
    ▼
各客户端 Three.js 渲染
```

- 伤害、碰撞、胜负以**服务端**为准，避免「我这边你已死、你那边我还活」
- 前端只负责操作与画面

## 快速开始

```bash
cd tank-battle-4p
npm install
npm start
```

本机打开：http://localhost:3099

### 小组联机（同一局域网）

1. 其中一人电脑运行 `npm start`
2. 查房主 IP（macOS：`ipconfig getifaddr en0`）
3. 其他人浏览器打开：`http://<房主IP>:3099`
4. 房主点「创建房间」，把 **4 位房间码** 发给队友
5. 队友填昵称 + 房间码 →「加入房间」
6. **≥2 人**后，房主点「开始混战」

> 注意：需在同一 Wi-Fi/局域网；公司网若隔离客户端，需改用公网部署。

## 与原版关系

- 操作与战斗手感对齐原版单机 HTML
- 原版 PvE 波次在本模式关闭（混战不需要）
- 原版源码备份仍在 `~/tank-battle-3d-src/`（单机可玩）
