# 对战规则（Agent / 人类观战）

- **模式**：单人合成（Suika 变体）。一名选手（人类或 Agent）掉落星球并合成。  
- **人类角色**：浏览器打开观战链接，观看 Agent 整局合成过程。  
- **无对手**：不存在第二名玩家；分数与合成链是唯一目标。

## 胜负与失败

- **无固定「通关」**：尽量高分；合成「太阳」为里程碑。  
- **失败**：任意已落地星球的顶部持续超过警戒线 `lineY` 达到约 **2.2 秒**。

## 合成与得分

- 两个 **相同等级** 星球碰撞 → 合成为 **下一级**。  
- 等级链：陨石 → 月球 → 水星 → 火星 → 金星 → 地球 → 海王星 → 天王星 → 土星 → 木星 → **太阳**。  
- 合成得分：`mergeScore(新等级索引) × combo`，其中 `mergeScore(i) = (i+1)*(i+2)/2`（i 从 0 起，合成出的新球 level）。  
- 连击：在 **1.5 秒** 内连续合成则 combo 递增。  
- **两个太阳**相遇 → **湮灭**，得分 `150 × combo`，场上两球消失。

## 掉落规则

- 随机只生成前 **5** 级（陨石～金星）作为新球。  
- 连续掉落冷却 **0.55 秒**。  
- Agent 须在 `canDrop === true` 时再 `drop`。

## 坐标

- 画布逻辑 **480×720**，原点左上，y 向下。  
- 左右墙约 x=10～470；地面 y≈706；警戒线 y≈132；出手高度 y≈72。
