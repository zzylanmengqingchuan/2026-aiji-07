# 第 07 天 · 2048 新手闯关

面向完全新手的 2048 网页游戏：

- **7 段动画演示**：讲清能移动 / 不能移动的特殊情况
- **5 个渐进关卡**：从 3×3 目标 64，到 5×5 目标 4096
- **WASD 操作** + 屏幕按键；游戏中高亮当前可走方向

## 怎么玩

1. 用浏览器打开 `index.html`
2. 点 **看动画教程**（推荐），看完 7 段演示
3. 进入关卡，从第 1 关开始闯关

### 操作（只用 WASD）

| 键 | 方向 |
|----|------|
| **W** | 上 |
| **A** | 左 |
| **S** | 下 |
| **D** | 右 |

也可用屏幕下方的 WASD 按钮，或手机在棋盘上滑动。  
**不用方向键 ↑↓←→。**

橙色按键 = 当前能走的方向；灰色 = 这个方向无效。

## 五个关卡

| 关卡 | 名称 | 棋盘 | 目标 | 说明 |
|------|------|------|------|------|
| 1 | 入门体验 | 3×3 | 64 | 最小棋盘，好上手 |
| 2 | 小试牛刀 | 4×4 | 256 | 进入经典尺寸 |
| 3 | 渐入佳境 | 4×4 | 512 | 开始需要布局 |
| 4 | 经典 2048 | 4×4 | 2048 | 原版目标 |
| 5 | 超越极限 | 5×5 | 4096 | 高手挑战 |

进度（通关、解锁、最佳分）保存在浏览器本地。

## 文件

- `index.html` — 页面结构
- `style.css` — 样式（手机优先）
- `game.js` — 游戏逻辑、动画教程与关卡
